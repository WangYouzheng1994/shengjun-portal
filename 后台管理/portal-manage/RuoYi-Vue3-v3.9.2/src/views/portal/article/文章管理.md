# 文章管理模块

## 功能概述
文章管理模块用于管理门户网站的各类文章内容，支持文章分类、发布状态管理等功能。

## 目录结构
```
article/
├── article/
│   └── index.vue    # 文章管理主页面
└── category/
    └── index.vue    # 文章分类管理
```

## 模块说明

### 1. 文章分类管理 (category)
- **功能说明**：管理文章的分类体系，支持树形结构展示
- **主要操作**：
  - 新增分类
  - 编辑分类信息
  - 删除分类
  - 分类启用/停用状态管理
- **查询条件**：分类名称、状态

### 2. 文章管理 (article)
- **功能说明**：管理具体的文章内容，支持富文本编辑
- **主要操作**：
  - 新增文章
  - 编辑文章内容
  - 删除文章
  - 文章发布/下架状态管理
- **查询条件**：标题、状态、分类

## 数据权限
- `portal:article:category:add` - 新增分类权限
- `portal:article:category:edit` - 编辑分类权限
- `portal:article:category:remove` - 删除分类权限
- `portal:article:article:add` - 新增文章权限
- `portal:article:article:edit` - 编辑文章权限
- `portal:article:article:remove` - 删除文章权限
