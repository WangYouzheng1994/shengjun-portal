# 产品管理模块

## 功能概述
产品管理模块用于管理门户网站的商品信息，包括产品分类、品牌、SKU规格、属性模板等完整的产品体系。

## 目录结构
```
product/
├── brand/
│   └── index.vue         # 品牌管理
├── category/
│   └── index.vue         # 产品分类管理
├── list/
│   ├── index.vue         # 产品列表管理
│   └── edit.vue          # 产品编辑
├── sku/
│   └── index.vue         # SKU规格管理
├── template/
│   └── index.vue         # 属性模板管理
└── 产品管理.md            # 本模块说明文档
```

## 模块说明

### 1. 产品分类管理 (category)
- **功能说明**：管理产品分类体系，支持树形结构展示
- **主要操作**：
  - 新增分类
  - 编辑分类信息
  - 删除分类
  - 分类启用/停用状态管理
  - 展开/折叠分类树
- **查询条件**：分类名称、状态
- **数据权限**：`portal:product:category:add`、`portal:product:category:edit`、`portal:product:category:remove`

### 2. 品牌管理 (brand)
- **功能说明**：管理产品品牌信息
- **主要操作**：
  - 新增品牌
  - 编辑品牌信息
  - 删除品牌
  - 品牌启用/停用状态管理
- **查询条件**：品牌名称、品牌编码、状态
- **数据权限**：`portal:product:brand:add`、`portal:product:brand:edit`、`portal:product:brand:remove`

### 3. 产品管理 (list)
- **功能说明**：管理产品主数据，维护产品基本信息
- **主要操作**：
  - 新增产品
  - 编辑产品信息
  - 删除产品
  - 产品上下架状态管理
- **查询条件**：产品名称/编码、所属分类、品牌、状态
- **数据权限**：`portal:product:list:add`、`portal:product:list:edit`、`portal:product:list:remove`

### 4. SKU规格管理 (sku)
- **功能说明**：管理产品的SKU（库存量单位）规格信息
- **主要操作**：
  - 编辑SKU信息
  - SKU启用/停用状态管理
- **查询条件**：产品名称/编码、SKU编码、状态
- **数据权限**：`portal:product:sku:edit`

### 5. 属性模板管理 (template)
- **功能说明**：管理产品属性模板，用于规范产品属性信息
- **主要操作**：
  - 新增模板
  - 编辑模板信息
  - 删除模板
  - 模板启用/停用状态管理
- **查询条件**：模板名称、状态
- **数据权限**：`portal:product:attrTemplate:add`、`portal:product:attrTemplate:edit`、`portal:product:attrTemplate:remove`

## 模块关系
```
产品分类 (category)
    │
    ├── 品牌 (brand)
    │
    └── 产品 (list) ─── 属性模板 (template)
            │
            └── SKU (sku)
```

## 产品数据流向
1. 先建立**产品分类**体系
2. 再创建**品牌**信息
3. 然后使用**属性模板**定义产品属性规范
4. 基于分类和品牌创建**产品**主数据
5. 最后为产品配置**SKU**规格信息
