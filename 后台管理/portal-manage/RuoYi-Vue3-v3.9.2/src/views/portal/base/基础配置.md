# 基础配置模块

## 功能概述
基础配置模块用于管理门户平台的核心配置信息，包括Banner图管理和公司信息管理。

## 目录结构
```
base/
├── banner/
│   └── index.vue    # Banner图管理
└── company/
    └── index.vue    # 公司信息管理
```

## 模块说明

### 1. Banner图管理 (banner)
- **功能说明**：管理门户首页的轮播图，支持图片上传、链接配置等
- **主要操作**：
  - 新增Banner图
  - 编辑Banner信息
  - 删除Banner图
  - Banner启用/停用状态管理
- **查询条件**：标题、状态
- **数据权限**：`portal:base:banner:add`、`portal:base:banner:edit`、`portal:base:banner:remove`

### 2. 公司信息管理 (company)
- **功能说明**：管理公司基本信息，如公司名称、简介、联系方式等
- **主要操作**：
  - 新增公司信息
  - 编辑公司信息
  - 删除公司信息
  - 公司启用/停用状态管理
- **查询条件**：公司名称、状态
- **数据权限**：`portal:base:company:add`、`portal:base:company:edit`、`portal:base:company:remove`
