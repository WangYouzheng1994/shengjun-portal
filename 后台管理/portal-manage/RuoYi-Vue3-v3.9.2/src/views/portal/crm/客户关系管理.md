# 客户关系管理模块

## 功能概述
客户关系管理模块用于管理客户信息、跟进记录、消息中心和客户标签等功能。

## 目录结构
```
crm/
├── customer/
│   └── list.vue     # 客户管理
├── followup/
│   └── list.vue     # 客户跟进记录
├── message/
│   ├── list.vue     # 消息管理
│   └── template.vue # 消息模板管理
└── tag/
    └── index.vue    # 客户标签管理
```

## 模块说明

### 1. 客户管理 (customer)
- **功能说明**：管理客户信息档案，记录客户基本资料和状态
- **主要操作**：
  - 新增客户
  - 编辑客户信息
  - 删除客户
  - 客户状态/等级管理
- **查询条件**：客户姓名、手机号码、公司名称、客户状态、客户等级
- **数据权限**：`portal:crm:customer:add`、`portal:crm:customer:edit`、`portal:crm:customer:remove`

### 2. 客户跟进记录 (followup)
- **功能说明**：记录与客户的每一次跟进信息，包括跟进方式、接触感受等
- **主要操作**：
  - 新增跟进记录
  - 编辑跟进记录
  - 删除跟进记录
  - 查看跟进历史
- **查询条件**：客户姓名、跟进方式、接触感受、跟进时间范围
- **数据权限**：`portal:crm:followup:add`、`portal:crm:followup:edit`、`portal:crm:followup:remove`

### 3. 消息管理 (message)
- **功能说明**：管理客户提交的消息/留言，支持消息处理状态跟踪
- **主要操作**：
  - 查看消息列表
  - 回复消息
  - 删除消息
  - 标记消息已读/未读
- **查询条件**：联系人、联系电话、处理状态、是否已读
- **数据权限**：`portal:crm:message:add`、`portal:crm:message:edit`、`portal:crm:message:remove`

### 4. 消息模板管理 (template)
- **功能说明**：管理消息回复模板，提高客服工作效率
- **主要操作**：
  - 新增模板
  - 编辑模板内容
  - 删除模板
  - 模板启用/停用
- **查询条件**：模板名称、模板类型、状态
- **数据权限**：`portal:crm:template:add`、`portal:crm:template:edit`、`portal:crm:template:remove`

### 5. 客户标签管理 (tag)
- **功能说明**：管理客户标签，用于客户分类和营销分析
- **主要操作**：
  - 新增标签
  - 编辑标签
  - 删除标签
  - 标签启用/停用
- **查询条件**：标签名称、状态
- **数据权限**：`portal:crm:tag:add`、`portal:crm:tag:edit`、`portal:crm:tag:remove`
