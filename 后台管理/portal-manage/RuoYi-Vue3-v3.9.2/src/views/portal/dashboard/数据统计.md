# 数据统计模块

## 功能概述
数据统计模块用于展示门户网站的各类运营数据指标，包括访问量、文章点击量等统计数据。

## 目录结构
```
dashboard/
└── index.vue    # 数据统计仪表盘
```

## 模块说明

### 数据统计仪表盘 (dashboard)
- **功能说明**：汇总展示门户网站的各项运营数据指标
- **主要指标**：
  - 今日点击量：实时统计当天的访问量
  - 昨日点击量：统计上一天的访问量
  - 本月文章点击量：统计当月文章的累计点击量
  - 数据趋势：展示与昨日/历史的环比变化
- **数据更新**：支持实时刷新，获取最新统计数据

## 相关接口
- 获取统计数据接口：提供今日访问量、历史访问量、环比趋势等数据
